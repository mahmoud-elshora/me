تعديل المحتوى الأصلي للصفحات

عدّل أي نص أو صورة موجودة في صفحات الموقع الستاتيك بدون لمس الكود.
اختر الصفحة، ثم نوع التعديل، ثم محدد العنصر (CSS Selector) مثل: #about h2 أو .hero-title أو section#initiative p:first-of-type، ثم القيمة الجديدة. تُطبق فوراً لكل الزوار.

يمكن استخدام أي محدد CSS صالح. افتح صفحة الموقع → كليك يمين → فحص لتعرف المحدد.
إذا العنصر <img> سيُغيّر src. وإلا ستوضع كخلفية.
